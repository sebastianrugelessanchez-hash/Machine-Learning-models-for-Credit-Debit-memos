PIPELINE DE PREDICCIÓN - CREDIT & DEBIT MEMOS (USA)
====================================================

A. Raw Data         → Datos transaccionales de SAP (pestaña Reference, 2020-2025).
B. Data Validation  → Validar tipos, rangos y distribuciones antes de procesar.
C. Data Cleaning    → Eliminar columnas innecesarias, castear tipos y tratar outliers.
D. Enrichment       → Merge con Stronghold info (SOrg+SOff+SGrp) para obtener Region y Stronghold. Mapear Dv a división.
E. Filter USA       → Conservar solo registros con Stronghold = US-ACM.
F. Target Eng.      → Separar Net value en credit_net_value y debit_net_value según SaTy.
G-H. Aggregation    → Agregar a nivel mensual: 1 fila = customer × region × division × mes.
I. Features         → Crear lags (1,3,12), rolling means (3,6) y features categóricas.
J-K. Model Select.  → Evaluar Lasso, LightGBM, XGBoost y CatBoost con walk-forward validation.
L-P. Train & Forecast → Entrenar modelo ganador con todo el histórico y predecir enero 2026.
Q. Reporting        → Unir resultados, rankear clientes y exportar para Finanzas.
